# **App Name**: DigitalLizard IPTV Landing

## Core Features:

- Hero Section: Visually engaging hero section with a cinema background, headline, and clear CTAs to encourage user engagement.
- Features Grid: Showcase key features of the IPTV service using an interactive grid layout that highlights the benefits such as instant activation and all-device compatibility.
- Device Compatibility Display: Display a comprehensive list of supported devices with recognizable logos that build user trust and indicate wide accessibility.
- Pricing Plans: Present tiered pricing plans clearly with feature comparisons and a highlighted 'Best Value' option to guide user purchasing decisions.
- FAQ Accordion: Provide an accordion-style FAQ section that answers common user queries about trials, device limits, buffering, and more, enhancing user trust and reducing support inquiries.
- Customer Testimonials: Showcase customer testimonials in a visually appealing chat-style layout to build trust and demonstrate service satisfaction.
- Blog Grid: Feature a blog grid that showcases recent tutorials and guides, helping users install and optimize the IPTV service on various devices.

## Style Guidelines:

- Primary color: Emerald green (#22C55E), embodying reliability and trust.
- Background color: Light emerald green (#E0F7E8), providing a gentle backdrop.
- Accent color: Forest green (#16A34A), used for calls to action.
- Headline font: 'Outfit' (sans-serif) for headings, providing a modern, structured feel.
- Body font: 'Inter' (sans-serif) for body text and UI, offering clarity and readability.
- Use lucide-react icons for general UI elements and react-icons for brand logos, ensuring a consistent and recognizable visual language.
- Employ soft shadows, rounded-xl borders, and glassmorphism effects to create a modern and visually appealing user interface.
- Apply subtle fade and slide-in animations with framer-motion, using a 150-250ms ease-out transition to create smooth, non-jarring interactions.
